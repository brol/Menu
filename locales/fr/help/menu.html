Menu vous permet d’afficher un menu pouvant avoir plusieurs niveaux sur votre blog, si votre thème le supporte.

Menu
Ici sont affichés les items de votre menu.
Vous pouvez les ordonner en déplaçant les lignes vers le haut ou vers le bas.
Vous pouvez également modifier le libellé, la description, l’URL et le niveau de chaque item de menu.
Une fois terminé, cliquez sur « Mettre à jour le menu ».
Supprimer les items de menu sélectionnés
Vous permet de supprimer les items de menu que vous souhaitez en cochant les cases pour chacun d’eux.
Ajouter un item de menu / Editer un item de menu
Indiquez ici le libellé, l'URL, la description (facultative), le niveau, la langue et la classe CSS de l’item de menu.
Une fois effectué cliquer sur « Enregistrer ».
  • Libellé de l'item de menu : indiquez ici le libellé. Par exemple "Archives". Ce champs est obligatoire.
  • URL de l'item de menu : indiquez ici la destination. Ce champ est obligatoire.
  • Description : indiquez ici une courte description pour cet item (sera affiché sous le libellé en fonction du thème sélectionné). Par exemple "Archives de 2004 à 2011".
  • Niveau : indiquez ici le niveau de l'item de menu. Pour un item de 1er niveau indiquez 1 dans la case, pour un item de 2ème niveau indiquez 2, etc. Indiquer 0 dans la case masquera l'item côté public.
  • Langue : indiquez ici la langue du lien.
  • Classe CSS : indiquez ici la classe CSS spécifique à l'item.
Intégration dans un thème
Le template {{tpl:Menu}} n'affichera par défaut sur le blog que les liens de niveau 1.
Si vous désirez un menu affichant les sous-menus, il faut le spécifier avec l'attribut "level" ainsi : {{tpl:Menu level="0"}}
Le widget lui affiche par défaut toute la hiérarchie du menu.
