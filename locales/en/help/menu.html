Menu allows you to display a menu can have multiple levels on your blog, if your theme supports it.

Menu
Here are displayed the items on your menu.
You can reorder this list by dragging its items up and down.
You can also change the wording, description, URL, and the level of each menu item.
Once done, click on Update menu.
Delete selected menu items
Allows you to delete the menu items you want by checking the boxes for each of them.
Add a menu item / Edit a menu item
Enter here the text, URL, description (optional), the level of language and the CSS class of the menu item.
Once done, click on Save.
  • Label of the menu item: the anchor text. For example "Archives". This field is mandatory.
  • URL of the menu item: the item URL. This field is mandatory.
  • Description: a short description for the menu item (will be displayed under the label depending on the selected theme). For example: "Archives from 2004 to 2011".
  • Level: here indicate the level of the menu item. For item 1 level indicate 1 in the box for an item in the second level indicate 2, etc. Indicate 0 in the box will hide the item public side.
  • Language: indicate here the language link.
  • CSS class: indicate here the specific CSS class to the item.
Integration into a theme
Template {{tpl:Menu}} will default on the blog that Level 1 links.
If you want a display submenus menu, specify it with the "level" attribute as follows: {{tpl:Menu level="0"}}
The widget displays default it all the menu hierarchy.
If the theme has not been planned to integrate the present plugin you must add in the root of the theme folder a file named: menu.nomdudossierdutheme.php
Take an example of an already existing file in the themes-config folder located in the Plugin Menu directory.
The menu.exemple-theme.php file contains all the instructions that can be given.
